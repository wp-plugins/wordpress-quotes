== Description ==

Show off the valuable, motivational quotes & impress your reader
This Simple WordPress Plugin displays a valuable, motivational, inspirational & successful quotes to your blog's sidebar.

= Features =

* **Simplicity -** No complex installation or settings; just add the widget to your sidebar and go - it's that simple.
* **Quality Quotes -** This Plugin has 365 valuable & motivational quotes integrated & more on regular updates
* **Double Options Display -** You can choose to update the quote daily or have a random quote on each page view.

== Installation ==

1. Upload the file `wordpressquotes.php` to the `/wp-content/plugins/Wordpress Quotes` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets and drag WordPress Quotes' to your sidebar
4. Update the settings in the widget: Title, When to Update, etc.

= Usage = 

1. Just add a WordPress-Quotes widget.
1. Enter the title.
1. Choose the update option.
1. Click save.

For usage, just see the screenshots. For detail usage and to use some inbuilt functions and features refer the [Documentation](http://www.ec-tech.info/web/wordpress-plugins/wordpress-quotes/) page in [EC-Technologies](http://www.ec-tech.info/)



== Frequently Asked Questions ==

Settings can be edited from within the widget itself, here are the options:

= "Title" Option: =
This option will change the front-end title of your widget, it defaults to "WordPress Quotes"

= "Update Quotes" Option: =
This option allows you to either update the quote once per day, or have it display a random quote each time your blog is viewed. 

= "Give Us Credit" Option: =
This option helps us promote our widget and we hope you'll keep it. However, to easily remove this line, simply choose "no" under the "Give Us Credit" option.
